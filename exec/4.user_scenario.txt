Testman에 오신 걸 환영합니다!
저희 Testman은 API 테스트와 부하 테스트 그리고 간단한 성능 테스트까지 종합적인 테스트가 가능합니다.
테스트는 Workspace별로 관리되어 한번의 URL 입력으로 다양한 테스트를 경험할 수 있습니다.

<시나리오>

1. 로그인

    사이트에 로그인 (testt / aa12345!) 합니다.


2. 프로젝트 생성

    API와 부하테스트를 위한 프로젝트를 생성합니다.

    - 우측 상단의 Add 버튼을 클릭하면 새로운 프로젝트를 생성할 수 있습니다.
    - 대상이 되는 Title, URL, 설명, 대표 이미지를 설정해 줍니다.
    - API 테스트 페이지로 이동합니다.
    - 페이지 상단의 Projects 버튼을 통해서도 이동 가능합니다.
    - 같은 프로젝트라도 local주소와 서버 주소로 워크스페이스를 만들어서 관리하며 테스트 합니다.
    - 워크스페이스를 만들 때 기본이 되는 url을 입력하면 탭을 만들 때마다 자동완성이 됩니다.


3. API 테스트 
    
    별도의 프로그램 설치나 코드에서의 설정 없이 API 테스트가 가능합니다.
    
    - 공통 된 URL 이외에 Path를 지정하고
    - HTTP METHOD를 고르고 전달할 값들을 Header와 Body로 나눠 입력합니다.
    - Send 버튼을 클릭하면 전송이 가능하고, 리턴되는 값을 확인할 수 있습니다.
    - 입력 값에 따른 API 결과를 JSON 포맷에 맞춰 표현해 줍니다.
    - 리턴된 값을 복사하기 쉽도록 COPY버튼이 활성화 됩니다.


4. 부하 테스트 
    
    우리 프로젝트가 동시에 몇명이 몇개의 접속까지 버틸 수 있을지 간단하게 확인할 수 있습니다.

    - 화면 우측의 LOAD 탭으로 변경해 줍니다.
    - API 테스트 입력값을 그대로 부하테스트에 적용할 수 있습니다.
    - 코드의 속도 비교 및 대략적인 트래픽 범위를 예상할 수 있습니다.
    

5. 성능 테스트
    
    프로젝트 페이지별로 성능 테스트가 가능합니다.

    - 페이지 상단의 PERFORMANCE 버튼을 클릭해 성능테스트 화면으로 이동합니다.
    - 페이지의 URI를 입력해주세요
    - 조금만 기다려 주시면 성능 테스트를 점수를 확인 가능합니다!
    - 성능 측정치의 절대값 및 상대값을 제공합니다.


6. 로그아웃

    사이트의 우측 상단 프로필애서 로그아웃을 클릭합니다.
